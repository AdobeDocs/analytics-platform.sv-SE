# Customer Journey Analytics värdplatser

Adobe Customer Journey Analytics ligger hos datacenter i företagsklass från leverantörer av offentliga molntjänster i Nordamerika, Europa och APAC.

Vid etableringen anger kunderna i vilken region deras Adobe Experience Platform-data kommer att finnas. De data som hämtas in till Customer Journey Analytics från Adobe Experience Platform datasjön lagras i samma region.

Mer information finns i [Regional datainsamling](https://experienceleague.adobe.com/sv/docs/core-services/interface/data-collection/rdc) i Adobe Experience Cloud-dokumentationen.


## Visa datacentret där data lagras

>[!NOTE]
>
>Det går inte att flytta data från ett datacenter till ett annat.

Så här visar du vilket datacenter dina data lagras i:

1. Logga in på [Adobe Experience Cloud](https://experience.adobe.com).

1. Välj **[!UICONTROL Experience Platform]** i appväljaren ![App](/help/assets/icons/Apps.svg) längst upp till höger i gränssnittet.

1. Regionkoden för ditt tilldelade datacenter visas i den övre högra delen av Experience Platform.

   ![Datacenter på Experience Platform hemsida](assets/experience-platform-data-center.png)

1. Använd följande tabell för att förstå vilket geografiskt område din regionkod är kopplad till:

   | Adobe regionkod | Molnleverantör | Geografisk region |
   |-------------------|-------|-------------------------------------------|
   | VA7 | Azure | USA (standard) |
   | VA6 | AWS | USA (på begäran) |
   | NLD2 | Azure | Nederländerna, Amsterdam |
   | CAN2 | Azure | Canada Central, Toronto |
   | AUS5 | Azure | Australia, Sydney |
   | GBR9 | Azure | Storbritannien, London |
   | IND2 | Azure | Indien |
   | CHE2 | Azure | Schweiz |
